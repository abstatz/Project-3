Austin Crime Analysis

The goal of this analysis is to review the crime data of Austin, Texas. The purpose to identify the key areas of downtown Austin by zip code and to see if there are any discernible crimes tied to specific areas.

We pulled data from the Austin's goverment portal for crime data and analyzed the quality of the data usinga SQL Database. We had to modify the data to correct for the incorrect coordinates we obtained by shifting the longitude by the standard error we calcualted. From here we narrowed down the zip codes to focus on downtown Austin as the surrouding suburbs were skewing our heatmap. Finally we decided to focus on the key crimes in their respective zip codes.

After assembling the dataset, we used a number of data tools to build maps displaying the crime data in a heatmap and used the new Java Library of Google Charts to display the types of crime by zip code.
